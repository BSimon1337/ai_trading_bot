# Limitations

- This experiment uses prebuilt dataset rows and model predictions, not full exchange order-book replay.
- Fill quality, latency, and slippage are not fully modeled in this matrix.
- Regime changes can reduce future performance relative to the test window.
- Results should be treated as comparative evidence across configs, not guaranteed future outcomes.
- Paper/live execution can diverge from offline simulation due to API timing and market microstructure.
